# How to Install

There are two options for getting started with Rhythm Cafe development.

If you are **only going to change things on the frontend** (i.e. the Rhythm Cafe
website itself), and not anything on the backend Python side, you can opt for
the **frontend-only workflow.** This option lets you try out frontend changes
on the actual Rhythm Cafe server. It's easier to set up and needs no extra
configuration.

>\>\>\> [**click here for the frontend-only workflow**](./FRONTEND_ONLY.md) <<<

Otherwise, if you need to change things on the backend as well, you will need
to set up the **full stack**. This will require you to set up a whole bunch of
things.

>\>\>\> [**click here for the full stack**](./FULL_INSTALL.md) <<<
