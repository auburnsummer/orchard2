# CSS Prefixes List

Every css file, with the exception of (`shared.css`) has a unique prefix.

To avoid collisions this page lists what they all are.

 - `.he`: Header
 - `.nf`: NotFound
 - `.ph`: PublisherHeader
 - `.ui`: Input
 - `.ub`: Button
 - `.ba`: _base
 - `.pr`: _base.profile
 - `.pu`: publisher
 - `.re`: publisher.discord_register
 - `.rn`: publisher.discord_register_.new
